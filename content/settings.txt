Project images are created using screely:

Large Image:
    - Screenshot size: 2290 x 1302 (@2x = 1145 x 651)
    - Output image size: 1483 x 900

    Settings:
    - Resize to height: 768
    - Shadow "ON"
    - Light/Dark mode to taste
    - Window type "plain"
    - Background color match the project color
    - Padding
        - vertical "50px"
        - horizontal "66px"

Small Image:
    - Screenshot size: 1600 x 1200 (@2x = 800 x 600)
    - Output image size: 1024 x 800

    Settings:
    - Resize to height: 768
    - Shadow "OFF"
    - Light/Dark mode to taste
    - Window type "plain"
    - Background doesn't matter
    - Padding
        - vertical "0"
        - horizontal "0"
